## EMAIL AUTOMATION

### SETUP INSTALLATION

#### clone repository
`git clone https://github.com/gatotbima1104/email-automation.git`

#### cd cloned repository
`cd path-to-cloned-repo`

#### make python env
`python -m venv venv`

#### activate env
`source venv/bin/activate`

#### install libraries
`python install requirements.txt`

#### run app.py
`python app.py`

## Enjoy it
